# Student Manager (Java CLI App)
Built as a small backend-style project to demonstrate Java fundamentals and file persistence.
A simple console-based student management system built in Java.
Features:
- Add students
- View students
- Data saved to file (students.txt)
- Input validation
- Error handling
Tech:
- Java
- File I/O
- IntelliJ IDEA
How to run:
1. Clone repo
2. Open in IntelliJ
3. Run StudentManager.java
Example:
1 → Add student
2 → View students
3 → Exit
